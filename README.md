# Student-Management-System
A desktop-based Student Management System built with Python, Tkinter, and SQLite3. Features a full CRUD (Create, Read, Update, Delete) workflow with search functionality and a persistent database backend.

# Student Management System (DBMS)

A lightweight and efficient desktop application designed to manage student records. This project demonstrates the integration of a Python-based Graphical User Interface (GUI) with an SQLite database to handle data persistence and management tasks.

## ✨ Features
* **Add Student:** Register new students with a unique ID, Name, and Age.
* **Search Functionality:** Quickly find students using either their Name or ID.
* **Delete Records:** Remove student data from the system via unique ID.
* **Data Persistence:** Uses SQLite3 to ensure data is saved even after the application is closed.
* **Input Validation:** Basic error handling for empty fields and duplicate IDs.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **GUI Framework:** Tkinter
* **Database:** SQLite3 (Serverless relational database)
